# 🎯 MPU6050 快速參考卡

## 座標軸方向（實際安裝）

```
鼓棒示意圖：

     握把端 ←── X 軸 ──→ 鼓棒頭
              (鼓棒長度)
                  
         Y 軸 (左右)
         ←─────┼─────→
                │
                │ Z 軸 (上下，敲擊方向)
                ↓
```

| 軸 | 方向 | 用途 |
|---|------|------|
| X | 鼓棒長度 | 前後移動 |
| Y | 左右 | 橫向擺動 |
| Z | 上下 | **敲擊動作** ⭐ |

## 角度定義

| 角度 | 公式 | 動作 |
|-----|------|------|
| **Roll** | `atan2(Y, Z)` | 左右傾斜 |
| **Pitch** | `atan2(-Z, √(X²+Y²))` | 上下擺動 |
| **Yaw** | 陀螺儀積分 | 水平旋轉 |

## 測試方法

| 動作 | Roll | Pitch | Yaw |
|-----|------|-------|-----|
| 向左傾 | ➕ | - | - |
| 向右傾 | ➖ | - | - |
| 向上抬 | - | ➕ | - |
| 向下壓 | - | ➖ | - |
| 順時針轉 | - | - | ➕ |
| 逆時針轉 | - | - | ➖ |

## 打擊偵測重點

- **主要軸**: Z 軸（上下方向）
- **閾值**: 2.0g
- **特徵**: Z 軸加速度劇烈變化

## 如果方向相反？

訪問: `http://localhost:5001/calibration`

勾選對應的反轉選項：
- ☑️ 反轉 Roll 軸
- ☑️ 反轉 Pitch 軸  
- ☑️ 反轉 Yaw 軸
